# Simple Calculator (Java)

## Description
This is a simple console-based calculator built in Java.  
It allows you to perform basic arithmetic operations: addition, subtraction, multiplication, and division.

## Features
- Add two numbers
- Subtract two numbers
- Multiply two numbers
- Divide two numbers (handles division by zero)
- User-friendly console interface
- Input validation to prevent invalid entries
- Results are displayed with 2 decimal places for readability

## Technologies Used
- Java
- Scanner for user input
- Basic OOP concepts
