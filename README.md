# Simple Calculator (Java)

## 📌 Description

This is a simple console-based calculator application built using Java.
It allows users to perform basic arithmetic operations in a menu-driven console interface.

---

## ✨ Features

* Addition of two numbers
* Subtraction of two numbers
* Multiplication of two numbers
* Division (with zero-division handling)
* Modulus operation (%)
* Square of a number
* Cube of a number
* Factorial calculation
* Input validation (prevents invalid inputs)
* Clean and user-friendly console interface
* Continuous menu until user exits

---

## 🛠 Technologies Used

* Java (Core Java)
* Scanner class for user input
* Loops and conditional statements
* Methods for code reusability

---

## ▶ How to Run

### 1. Open terminal in project folder

Make sure you are inside the folder where the file is saved.

### 2. Compile the program

```bash
javac Calculator.java
```

### 3. Run the program

```bash
java Calculator
```
